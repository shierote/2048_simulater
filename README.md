# ゲーム2048のシミュレーション
## 実行方法
クローンしたのちに
複数回試したければ
```
$ git clone git@github.com:shierote/2048_simulater.git
$ cd 2048_simulater
$ ruby simulater.rb --game_num 100 --hist --alg tree3 --log
```
game_numの後ろでシミュレーションしたいゲーム回数、algの後ろで使いたいアルゴリズムを指定できる。
