# ゲーム2048のシミュレーション
## 実行方法
クローンしたのちに
複数回試したければ
```
$ cd 2048_simulater
$ ruby simulater.rb
```

局所的に試したければ
```
$ cd 2048_simulater
$ ruby test.rb
```
